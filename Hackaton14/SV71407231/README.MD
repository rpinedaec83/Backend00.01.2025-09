## Comandos especiales
#### /clear
```
Borra todo el historial de mensajes
Elimina los mensajes tanto del chat como de la base de datos
Se sincroniza automáticamente en todos los clientes conectados
```
#### /bot <mensaje>
```
Envía un mensaje al asistente IA
El bot responde automáticamente en el chat
La respuesta también se guarda en la base de datos
```
#### Edición y eliminación de mensajes

```
Para editar un mensaje, se necesita dar clic en EditarMsg
```
